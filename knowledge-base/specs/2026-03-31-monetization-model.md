# 付费模型设计

> 日期：2026-03-31

## 模型：Freemium + 按次额度混合

### 免费层

| 功能 | 说明 |
|------|------|
| 编辑 & 导出 PDF | 永久免费，无水印 |
| 模板 | 仅 1 套模板，不区分免费/付费模板 |
| 简历数量 | 1 份 |
| AI 功能 | 注册赠送 10 次额度 |

### 付费层

| 功能 | 说明 |
|------|------|
| 多份简历 | 超出 1 份后需付费解锁 |
| AI 额度 | 免费额度用完后，按次购买或订阅无限用 |

### 核心付费点

1. **简历数量限制** — 后端检查用户已有简历数量，超出限制时前端引导付费
2. **AI 调用额度** — 每次调用扣减额度，额度不足时前端引导充值/订阅

### 不做的方向

- 不做高级模板（保持单模板）
- 不做水印或降级导出（免费导出也是完整 PDF）

## 实现要点

- 后端：用户简历数量校验 + AI 额度计数（扣减 & 充值）
- 前端：限制提示弹窗 + 付费引导页
- 支付接入：待定（微信支付 / 支付宝）
