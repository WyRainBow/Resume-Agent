# Workspace 左侧「设置抽屉」面板方案

> 状态:设计提案(待拍板)｜分支:feature/template-market｜日期:2026-07-09
> 关联:`knowledge-base/specs/2026-07-08-workspace接入builder模板系统-design.md`、`2026-07-08-template-selector-grouping-design.md`

## 1. 要解决的三个问题

当前 `EditPreviewLayout` 三列:SidePanel **固定 300px 不可拖** + EditPanel 700px 可拖 + Preview flex-1。
SidePanel 这一条 300px 竖着叠了三块:**模块列表 + 模板画廊(8 缩略图) + 排版面板**。

| 问题 | 现状 | 根因 |
|---|---|---|
| 太挤 | 排版滑块/选项塞进 ~268px 可用宽 | 一条窄轨干三件事,竖叠 + 滚动 |
| 模板不突出 | 缩略图 ~56px,`flex-wrap` 每行 4 个 | 窄轨放不下大图,看不清版式差异 |
| 排版没横向空间 | 页边距/行距/字号滑块被压短 | 同上,268px 摆不开 |

## 2. 核心方案:左侧「设置抽屉」(Settings Drawer)

一句话:轨道默认只留**模块导航 + 一条设置触发条**;点「模板 / 排版」→ 一个 **~560px 宽抽屉从轨道右缘 slide 出**,盖住模块轨 + EditPanel 左半,**预览(flex-1)始终露出**——边调边看右侧预览。收起→缩回轨道。

为什么"盖住 EditPanel"在这个场景是对的:调模板/排版时,你盯的是**预览**,不是左边的编辑表单。所以盖掉表单不影响,反而把横向空间还给了设置。

## 3. 交互细节

- **触发**:模块导航顶部一条 segmented「🎨 模板 ｜ ⚙ 排版」。点一个 → 抽屉从轨道右缘 slide 出(~200ms ease-out),segmented 变成抽屉头部的 tab。
- **收起**:点 ✕ / 点 scrim / 再点当前 tab → 抽屉 slide 回。overlay 方式**无 reflow**,EditPanel 原样复现。
- **切 tab**:模板 ↔ 排版,抽屉宽度不变,只换内容,即时切(无 slide)。
- **选模板**:点大缩略图 → 立即 `applyTemplatePreset` + 切 `templateType`,预览刷新;**抽屉不自动关**(方便连续试不同模板),靠 ✕ 手动收。
- **调排版**:滑块/选项即时写 `builderSettings`(html)或 latex 设置,预览实时刷新——复用现有 `FormattingControls` 的 onChange,不新增链路。

## 4. 两个 tab 的内容

**模板 tab**
- 2 列大缩略图(每个 ~150×200,是现在 56px 的约 3 倍),分组标题「原生 LaTeX」「在线 HTML」。
- 当前项黑框高亮 + 角标 ✓。
- 复用 `Builder/components/TemplateSelector` 的缩略图,仅放大尺寸 + 改 2 列 grid。8 个选项(Classic LaTeX + 7 HTML)。

**排版 tab**
- 直接放现有 `FormattingControls`,在 ~520px 宽里:滑块整条铺开、档位按钮成排、字体下拉全宽。
- 按 `templateType` 分支:latex 模板显示 latex 那套(字号/边距/行距);html 模板显示 `builderSettings` 那套。沿用现有分支,不改契约。

## 5. 组件 / 文件改动映射(实现阶段;当前仅方案)

| 文件 | 改动 |
|---|---|
| `Workspace/v2/EditPreviewLayout.tsx` | 加 `settingsDrawer: null｜'template'｜'format'` 状态;抽屉容器(`absolute left-0 w-[560px]` z 高于 EditPanel + scrim);轨道宽 300→280 |
| `Workspace/v2/SidePanel/index.tsx` | 拆分:模块列表留原位;**模板选择 + 排版设置移进抽屉**;顶部加 segmented 触发条 |

复用为主,基本不新增组件——把已有的两块搬进抽屉、放大模板缩略图而已。

## 6. 已定决策(2026-07-09 看草图后确认)

1. **盖住方式**:✅ overlay + scrim(抽屉浮在 EditPanel 上,盖住区加半透明遮罩,无 reflow)。
2. **触发位置**:✅ 模块轨顶部 segmented「🎨 模板 ｜ ⚙ 排版」。
3. **选模板后**:✅ 不自动关,靠 ✕ / 点遮罩手动收(方便连续试不同模板)。

## 7. 备选方案(未采用,记录备查)

- **B1 · 排版抽屉挂到预览侧**:排版按钮放预览工具栏,面板从右侧滑出贴着预览。优点:设置紧挨它作用的对象。缺点:模板/排版落两处,心智不统一。→ 若你更看重"设置贴预览",可把**排版 tab 单独**改成右侧抽屉。
- **B2 · 模板用居中模态画廊**:模板做成全屏模态大画廊。优点:模板最大最突出。缺点:模态比抽屉重,和排版又不同形态。
- 本方案(左侧抽屉 + 双 tab)取的是**单一位置、单一形态、复用最多**的折中。
