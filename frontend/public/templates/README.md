# 模板预览图片说明

## 需要的图片文件

请将以下模板预览图片放置在此目录下：

1. `classic.png` - 经典模板预览图
2. `general.png` - 通用模板预览图
3. `custom.png` - 自定义模板预览图
4. `elegant-blue.png` - 雅蓝模板预览图
5. `modern.png` - 现代模板预览图
6. `fresh.png` - 清新模板预览图
7. `professional.png` - 专业模板预览图
8. `creative.png` - 创意模板预览图
9. `minimalist.png` - 极简模板预览图

## 图片规格建议

- **尺寸**: 建议 400x300px 或 800x600px
- **格式**: PNG 或 JPG
- **内容**: 简历模板的预览截图，展示模板的整体布局和样式

## 临时占位符

如果图片尚未准备好，系统会自动显示默认图标作为占位符。
