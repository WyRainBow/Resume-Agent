# UP简历平台 (upcv.tech) 调研分析报告

> 调研日期: 2025-12-27
> 平台地址: https://upcv.tech
> 调研目的: 为实现基础对话式简历生成 Agent 提供设计参考

---

## 一、平台概述

**UP简历** 是一款基于 AI 技术的免费在线简历制作工具，主打 AI 智能生成专业简历内容。核心功能包括：

- AI 智能创建简历
- 导入现有简历
- 在线编辑
- 多种模板
- 中英文切换
- PDF 导出

---

## 二、核心功能模块分析

### 2.1 简历管理界面 (Dashboard)

**布局结构**：左侧导航栏 + 右侧主内容区

| 区域 | 功能 | 设计特点 |
|------|------|---------|
| **左侧导航** | 全局功能入口 | 垂直列表 + 图标 + 文字，固定宽度约200px |
| **主内容区** | 核心操作区 | 占据约80%宽度，包含标题、操作区、辅助信息 |

**导航模块**：
- 简历（核心功能）
- UP AI（AI 辅助功能）
- 用户反馈
- 用户信息（底部显示）

---

### 2.2 AI 创建简历流程

#### 阶段 1: 选择创建方式

界面提供两个卡片式选项：

```
┌─────────────────────────┐  ┌─────────────────────────┐
│      [AI 图标]           │  │      [下载 图标]         │
│   AI 智能创建            │  │   导入现有简历           │
│ AI 帮你快速生成精美简历  │  │ 支持导入多种格式文件     │
└─────────────────────────┘  └─────────────────────────┘
```

**设计特点**：
- 圆角卡片 + 浅灰色边框
- hover 时有阴影或背景色变化
- 图标采用品牌蓝色填充

#### 阶段 2: 对话式引导创建

**核心创新**：左右分栏 + 实时预览

```
┌──────────────────────┬──────────────────────┐
│   对话引导区          │   简历预览区          │
│                      │                      │
│  [AI 头像]           │  ┌─────────────────┐ │
│  让我们开始吧        │  │ 个人总结         │ │
│  [学生] [职场人士]   │  │ 教育经历         │ │
│                      │  │ 实习经历         │ │
│  [AI 头像]           │  │ 项目经历         │ │
│  分享你的求学经历    │  │ 社团组织         │ │
│  [输入框]            │  │                 │ │
└──────────────────────┴──────────────────────┘
```

**对话流程**：

1. **身份确认** → 选择"学生"或"职场人士"
2. **教育背景引导** → 输入学校名称、专业、学历
3. **经历填充** → 实习经历、项目经历、社团组织
4. **确认生成** → 查看完整简历预览

#### 阶段 3: AI 帮写功能

**核心特性**：
- 文本框右上角有"AI 帮写"按钮（紫色 + 星星图标）
- 提供"写作公式"提示：
  - 【主修课程 + 成绩】→【获得奖项 + 排名】→【学生工作/科研/社团活动】
- 点击后 AI 自动生成内容，用户可编辑修改

- 项目名称（突出显示）

- 项目链接/附件（可选）

---

### 2.4 交互设计特点

#### 1. 实时预览联动
- 左侧输入信息立即同步到右侧预览区
- 用户可直观看到简历效果
- 避免"填完再改"的繁琐

#### 2. 模块化引导
- 将简历创建拆分为多个小步骤
- 每一步都有明确引导
- 降低用户认知负荷

#### 3. 情感化交互
- AI 语言友好、鼓励（如"太棒了！✨"）
- 使用 emoji 增加亲和力
- 缓解用户填写简历的焦虑

#### 4. 操作按钮设计

| 操作 | 颜色 | 样式 |
|------|------|------|
| 保存/提交 | 蓝色 (#6c8cff) | 主要操作 |
| 删除 | 红色 | 危险操作 |
| 取消 | 灰色 | 次要操作 |

## 六、实现优先级建议

### Phase 1: 基础对话 Agent (MVP)

1. **对话状态管理**
   - 实现 `ConversationState` 类
   - 定义对话步骤流程
   - 支持上下文记忆

2. **前端对话界面**
   - 左右分栏布局
   - 消息气泡组件
   - 实时预览联动

3. **后端 Agent 逻辑**
   - 消息解析
   - 信息提取
   - 下一步引导生成

### Phase 2: AI 帮写功能

1. **AI 帮写按钮**
   - 集成到各模块输入框
   - 根据上下文生成内容

2. **写作公式提示**
   - 展示内容结构建议
   - 降低用户输入门槛

### Phase 3: 优化增强

1. **智能补全**
   - 基于已有信息推断缺失内容
   - 提供快捷选项

2. **个性化推荐**
   - 根据职位类型调整内容风格
   - 推荐相关技能/经历

**调研结论**：

UP简历的核心创新在于**对话式引导 + 实时预览 + AI 帮写**三位一体的体验设计。我们项目已有 AI 导入和实时预览能力，只需增加**对话式引导**和**AI 帮写**功能即可实现类似的体验。

建议先实现 **Phase 1 基础对话 Agent**，通过多轮对话收集信息并实时生成简历，验证核心价值后再逐步完善。
